# ARCH

> **Note:** This repository was created entirely using an Agent in the **Antigravity IDE** powered by **Gemini 3.0 Pro - High**. I did not code a single line manually.

## Extinct Species Catalog

ARCH is an interactive web application that serves as a catalog for extinct species. It allows users to explore various prehistoric eras and learn about the fascinating creatures that once roamed the Earth.

### Features

-   **Era-based Navigation:** Explore species by geological eras (e.g., Cretaceous, Jurassic, Pleistocene).
-   **Detailed Species Profiles:** View high-quality images, descriptions, and fun facts about each species.
-   **Discovery Info:** Learn about who discovered each species, when, and where.
-   **Responsive Design:** Optimized for viewing on different devices.

### Tech Stack

-   **React**: Frontend library for building the user interface.
-   **Vite**: Fast build tool and development server.
-   **React Router**: For client-side routing.
-   **CSS**: Custom styling for a modern and premium look.
